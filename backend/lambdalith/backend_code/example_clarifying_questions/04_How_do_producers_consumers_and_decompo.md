# Clarifying Questions — Example 4

**Grade / Subject:** 7th grade Life Science

**Student Question:**
> How do producers, consumers, and decomposers interact in a food web?

**Skill Being Tested:** Describe how energy and nutrients flow from producers through consumers and are recycled back by decomposers in an ecosystem.

## Clarifying Questions and Tutor Responses

**Q:** What is a food web?
**A:** A food web is a diagram that shows all the different feeding relationships and connections between various organisms in an ecosystem. You can think of it like a map made of many overlapping food chains that are all linked together.

**Q:** Should I use specific examples?
**A:** Yes, using specific examples of organisms you might find in a local ecosystem is a great idea! It will help make your explanation of how they work together much clearer.

**Q:** Is a diagram required?
**A:** A diagram is not required, but you are welcome to use one if it helps you explain your ideas. A written description works perfectly well too!

**Q:** Should I write a paragraph?
**A:** A paragraph is a great way to explain your thoughts, or you can use a few clear sentences if that's easier. As long as you address all the parts of the question, choose the format that helps you explain your ideas best!

**Q:** How many interactions are needed?
**A:** You should describe enough interactions to show how all three groups—producers, consumers, and decomposers—connect to each other. Your goal is to explain the full cycle that links them all together.

**Q:** Define 'interact' for this task.
**A:** In this context, 'interact' means to describe the ways these different groups affect, relate to, or behave toward one another. Think about the connections and relationships that exist between them in their shared environment.
