# Clarifying Questions — Example 2

**Grade / Subject:** 7th grade Life Science

**Student Question:**
> What is the difference between active transport and passive transport across a cell membrane?

## Generated Clarifying Questions

- What does 'active' mean?
- What does 'passive' mean?
- Should I include examples?
- How many differences?
- Should I draw a diagram?
- What is 'transport'?
